## License Keys

QShield uses offline HMAC-based license validation. License keys are generated by Homatch AI and validated locally â€” no phone-home required.

### Key Format
```
QS-{TIER}-{PAYLOAD}-{SIGNATURE}
```

Example: `QS-PRO-abc123def456-a1b2c3d4e5f6`

### Activation
1. Purchase a license at https://qshield.app/pricing
2. Receive license key via email
3. Open QShield Desktop > Settings > License
4. Enter license key > Activate

### Offline Validation
License keys are validated using HMAC-SHA256 without contacting any server. The key encodes:
- Tier (Free/Pro/Business/Enterprise)
- Expiry date
- Device limit
- Cryptographic signature

This means QShield works fully offline after activation.
